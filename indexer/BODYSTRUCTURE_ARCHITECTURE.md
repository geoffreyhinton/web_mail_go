## Key Architecture Components

### 1. **Content Type Router**
- Detects MIME type from parsed headers
- Routes to appropriate processor (text, multipart, attachment, RFC822)
- Handles edge cases and defaults gracefully

### 2. **Field Generation Engine**
- **Basic Fields**: Type, subtype, parameters, ID, description, encoding, size
- **Extension Fields**: MD5, disposition, language, location
- **Specialized Fields**: Line count for text, envelope for RFC822

### 3. **Structure Builders**
- **Text Processor**: Adds line count, handles charset parameters
- **Multipart Processor**: Recursive child processing, boundary handling
- **Attachment Processor**: Binary content metadata
- **RFC822 Processor**: Envelope generation, embedded message structures

### 4. **IMAP Compliance Layer**
- **BODYSTRUCTURE**: Full structure with extensions
- **BODY**: Simplified structure without extensions
- **Case Handling**: Upper/lowercase field names
- **NIL Handling**: Proper null value representation

### 5. **Serialization Engine**
- Converts Go structures to IMAP parenthesized format
- Type-safe serialization with proper quoting
- Recursive array handling for nested structures
